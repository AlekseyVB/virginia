# Virginia — Carpet Vision Toolkit

Набор настольных инструментов и ML-экспериментов для подготовки датасетов,
разметки изображений и визуальной проверки сегментации ковров на видео.

> Статус: исследовательский прототип. Репозиторий содержит полезные рабочие
> компоненты, но пока не является единым устанавливаемым приложением.

## Возможности

- ручная прямоугольная разметка объектов в GUI FORA;
- сохранение разметки в таблицу Pandas;
- преобразование координат FORA в формат YOLO;
- извлечение кадров из видео через FFmpeg;
- U-Net-сегментация ковров на видеокадрах;
- просмотр предсказаний и межкадровых различий в настольном интерфейсе.

## Установка

Проект ориентирован на Windows и настольный интерфейс.

```bash
python -m venv .venv
.venv\Scripts\activate
python -m pip install -r requirements.txt
```

Для `main.py` также требуется установленный системный FFmpeg, доступный в
`PATH`. Для сегментации необходим файл весов:

```text
Models_Weights/unet_carpets_weights_grayscale.h5
```

Весов в репозитории нет — их нужно предоставить отдельно или обучить модель.

## Запуск

Актуальный GUI-разметчик:

```bash
python "FORA aka CatsSquares v1.1.1.py"
```

Конвертер разметки:

```bash
python FORA_to_YOLO_DataConverter.py
```

Просмотр сегментации:

```bash
python Frames_Predictions_Player_X4_grayscale.py
```

## Текущие ограничения

- названия и пути некоторых входных файлов заданы непосредственно в коде;
- веса U-Net и демонстрационное видео не включены;
- старые версии GUI сохранены рядом с актуальной;
- отсутствуют автоматические тесты и единый CLI;
- `CLASS_NUM` в конвертере YOLO задаётся вручную;
- версии библиотек исходного запуска не зафиксированы.

## Авторство и лицензия

В исходных файлах встречаются материалы нескольких авторов. Перед повторным
распространением проекта следует подтвердить авторство компонентов и выбрать
лицензию репозитория. До добавления лицензии стандартное право на копирование
и модификацию автоматически не предоставляется.
